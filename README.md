# 🎰 Casino Simulator

Egy modern, böngészőben futó kaszinó-szimulátor virtuális zsetonokkal.

⚠️ Ez a projekt kizárólag szimulációs és fejlesztési célokat szolgál.  
Nincs benne valódi pénzes játék vagy kifizetés.

---

## ✨ Funkciók

### 🎰 Játékok

- 🎰 Slot Machine
- ♠ Blackjack
- 🎡 Rulett

### 👤 Profil rendszer

- Játékos név
- Automatikus mentés
- Statisztikák

### 📊 Statisztika

- Összes játék
- Nyerések száma
- Vereségek száma
- Összes nyert zseton
- Legnagyobb nyeremény

### 💾 Mentés

A projekt LocalStorage segítségével menti:

- zseton egyenleg
- játékos profil
- statisztikák
